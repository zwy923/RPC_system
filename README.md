# RPC_system

## Quick start

### Install dependency

```
cd client && npm install
```
back to root directory
```
cd server && npm install
```
back to root directory
```
cd server02 && npm install
```


The following command is entered in the root directory

#### Start client

``
npm run dev:client
``

#### Start server
Main server
```
npm run dev:server
```
Second Server
```
npm run dev:server2
```
